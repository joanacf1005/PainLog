## PainLog

PainLog is an application for recording pain episodes, related medication, and links between them, allowing users to create, read, update, and delete structured pain entries. The goal is to support symptom tracking over time by centralizing the information in a secure API with Supabase authentication.

It helps monitor pain, medication, and health patterns over time. By organizing this data in one system, PainLog can support more informed health tracking and decision-making.
